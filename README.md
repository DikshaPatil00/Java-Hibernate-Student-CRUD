# Java-Hibernate-Student-CRUD
A lightweight Java Backend application demonstrating full CRUD (Create, Read, Update, Delete) operations using the Hibernate ORM framework. This project serves as a practical implementation of database persistence and entity mapping in Java.

# Features

* Create: Add new student records to the MySQL database.
* Read: Fetch all students or search for a specific student by ID.
* Update: Modify existing student details (Name, Email, Course, etc.).
* Delete: Remove student records securely from the system.
* Object-Relational Mapping: Clean mapping between Java POJOs and Database tables.
  
# Tech Used
* Language: Java 17 (or your version)
* Framework: Hibernate 5.x / 6.x
* Database: MySQL
* Build Tool: Maven / Ant
* IDE: Eclipse
